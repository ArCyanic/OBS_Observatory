# FrameWork
A front-end Visualization Vue3 framework element-plus which has built a page framework.

# Details
- ElementPlus with auto-import(include icons)
- vue-router with start file
- pinia with start file
- d3, echarts
- typescript
- default layout
- proxy and '@' alias

# Use
- npm i
- npm run dev
